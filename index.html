Leaflet

Aan de rechterkant is een MapLibre kaart te vinden. Op deze kaart heb ik een GeoJSON laag geladen waarin ik Hallo heb geschreven met een aantal punten.

Deze data laat verder niet veel zien, maar ik vond het wel grappig om dit zo op een kaart te kunnen schrijven. Daarbij heb ik niet enorm veel research gedaan naar MapLibre waardoor ik niet echt wist hoe ik makkelijk de achtergrondkaart moest veranderen. Waar deze gekleurde vlakken met dikke witte lijnen voor kunnen worden gebruikt is voor mij nog steeds een groot raadsel.

MapLibre

Op de kaart aan de linkerkant is Leaflet gebruikt. Op deze kaart zijn meerdere datasets geladen. Zo is er een WFS laag op te vinden, drie GeoJSON punten, een GeoJSON polygoon, Leaflet markers met daarbij een polygoon die ze allemaal verbindt, een Leaflet cirkel en als laatste geodata van het nationaalgeoregister die automatisch worden ingeladen wanneer je op de knoppen klikt. Tevens kan je de GeoJSON punten en de WMS laag aan en uit zetten door middel van de bijbehorende knoppen. Als laatste zit er eigenlijk ook nog een WMS laag op deze kaart, echter wordt deze gehost via een lokale server waardoor deze niet kan worden laten zien.

Cesium

Hierboven zijn twee kaarten te zien. De linkerkaart is van Kepler. Kepler is een tool waarmee je data kan inladen en direct op de kaart kan visualiseren. Deze heb ik op de website gezet door het te embedden in een iFrame. Dit zorgt er dus eigenlijk voor dat er een website in mijn website wordt geladen. Als vierde en laatste kaart heb ik met javascript Cesium op mijn website gezet. Het is dus een javascript API die een 3D map viewer inlaadt. Op het moment staat deze ingesteld op Disneyland Parijs omdat ik daar erg vaak kom en het een leuke plek vind! :)

Kepler

Hieronder zijn twee datavisualisaties te vinden. De linker heb ik een tijdje terug gemaakt om aan Myrthe uit te leggen welke mensen om mij heen tot welke groep behoren. Ik vond het leuk om dit te visualiseren op deze manier omdat je nu op de groepsnamen kan klikken en in één oogopslag kan zien wie er bij die groep horen. In de lijst van datavisualisatie tools zag ik flourish en dacht 'die wil ik op mijn site hebben', niet wetende dat ik dus al een account had. De andere datavisualisatie die ik heb gebruikt is van de website Infogram. Infogram heeft mooie visualisatie tools die het ook direct voor je animeren, dit heb ik dus uitgetest met een enorm simpele dataset van de hoeveelheid films die ik in coronatijd heb gezien. In 2019 waren dit er nog 12, terwijl het er in 2020 nog maar 3 waren na het sluiten van de bioscopen. Ze zijn beiden met javascript ingeladen, dus met Flourish.js en infogram.js.
